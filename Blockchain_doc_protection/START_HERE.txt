===============================================================================
   ⚠️ COMMON ISSUES & SOLUTIONS
===============================================================================

"Python not found"
  → Install Python 3.8+ from https://python.org
  → Make sure to check "Add to PATH" during installation

"Node.js not found"
  → Install Node.js 18+ from https://nodejs.org
  → Restart command prompt after installation

"Truffle not found"
  → Run: npm install -g truffle
  → Restart command prompt

"IPFS not connecting"
  → Install IPFS Desktop: winget install --id IPFS.IPFS-Desktop -e
  → Launch IPFS Desktop before running script

"Port already in use"
  → Run: stop_everything.bat
  → Wait 5 seconds, try again

"Unicode/Emoji errors"
  → Scripts now handle this automatically
  → If issue persists, run as Administrator
