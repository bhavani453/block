                      🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════

Problem: "Python not found"
Solution: Install Python 3.8+ from https://python.org
          Make sure to check "Add to PATH" during install

Problem: "Node.js not found"
Solution: Install Node.js 18+ from https://nodejs.org
          Restart command prompt after install

Problem: "Truffle not found"
Solution: Run: npm install -g truffle
          Restart command prompt

Problem: "IPFS not connecting"
Solution: Install IPFS Desktop:
          winget install --id IPFS.IPFS-Desktop -e
          Launch IPFS Desktop before running script

Problem: "Port already in use"
Solution: Run stop_everything.bat
          Wait 5 seconds
          Try again

Problem: Services won't start
Solution: 1. Run check_status.bat to see what's wrong
          2. Check the service windows for errors
          3. Make sure no antivirus is blocking
          4. Run command prompt as Administrator
